# Praça Inteligente

Projeto acadêmico do primeiro semestre de engenharia da computação junto com engenharia civil e inteligência artificial.

## 📱 Sobre o Projeto

Site responsivo desenvolvido para o projeto "Praça Inteligente", permitindo acesso via QR code para fornecer informações e serviços inteligentes aos visitantes da praça.

## 🚀 Funcionalidades

- **Design Responsivo**: Funciona perfeitamente em desktop, tablet e dispositivos móveis
- **Acesso via QR Code**: URL otimizada para fácil acesso através de códigos QR
- **Navegação Intuitiva**: Menu moderno com animações suaves
- **Informações em Tempo Real**: Eventos, horários e informações úteis
- **Formulário de Contato**: Canal direto para feedback e sugestões
- **Animações e Interações**: Experiência de usuário moderna e agradável

## 📱 Como Acessar via QR Code

1. **Gerar QR Code**: Use qualquer gerador de QR Code online
2. **URL**: Aponte o QR Code para a URL onde o site estiver hospedado
3. **Escanear**: Usuários escaneiam o QR Code com seus smartphones
4. **Acesso**: Site abre automaticamente no navegador do dispositivo

## 🌐 Hospedagem Recomendada

Para facilitar o acesso via QR Code, hospede o site em:

- **GitHub Pages**: Gratuito e fácil de configurar
- **Netlify**: Deploy automático e domínio personalizado
- **Vercel**: Performance otimizada
- **Firebase Hosting**: Integrado com o ecossistema Google

## 📋 Requisitos de Hospedagem

- Servidor web estático (Apache, Nginx, etc.)
- Suporte a HTTPS (recomendado para QR Codes)
- Sem necessidade de backend ou banco de dados

## 🔧 Como Usar Localmente

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/seu-usuario/projeto-Praca-Inteligente.git
   cd projeto-Praca-Inteligente
   ```

2. **Abra o site**:
   - Dê duplo clique no arquivo `index.html`
   - Ou use um servidor local:
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Node.js (com http-server)
     npx http-server
     ```

3. **Acesse**: Abra `http://localhost:8000` no navegador
